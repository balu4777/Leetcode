# LeetCode Solutions in C#

A clean, structured, and scalable repository for solving LeetCode problems using **C#**.

## ?? Features
- ? Organized by DSA topic (Arrays, Strings, Trees, DP, etc.)
- ? Clean C# class-based solutions
- ? xUnit test cases for every problem
- ? Helper models (ListNode, TreeNode, GraphNode)
- ? Utility builders for trees & linked lists
- ? Easy to navigate & extend
